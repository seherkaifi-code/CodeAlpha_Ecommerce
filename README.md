# 🛒 E-commerce Web Application

## 📌 Description
This is a simple full-stack e-commerce website developed as part of an internship task. The application allows users to view products, add items to a cart, and perform login functionality.

## 🚀 Features
- Product listing
- Add to cart system
- User login
- Dynamic frontend

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## ▶️ How to Run
1. Install Node.js
2. Run: npm install
3. Run: node server.js
4. Open: http://localhost:3000

## 📚 Learning Outcome
Learned full-stack development and API integration.
